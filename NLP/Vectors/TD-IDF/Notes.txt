TF-IDF [Term frequency - Inverse Document frequency]
This is another techinique to convert word to vector
This techinue is better techinue compared to BOW.

Below formula will be used word conversion

TF = Num of repeated word in sentence/Num words in the sentence

IDF = loge[Num of the sentences/Num of sentences containing the word]

Let see how we can convert below 3 sentences to vector
s1 = good boy
s2 = good girl
s3 = both boy girl good

TF = Num of repeated word in sentence/Num words in the sentence
         s1         s2         s3
good    1/2         1/2         1/3
boy     1/2         0           1/3
girl    0           1/2         1/3
both    0           0           1/3

IDF = loge[Num of the sentences/Num of sentences containing the word]
         s1         s2          s3
good    loge(3/3)=0 loge(3/3)=0 loge(3/3)=0
boy     loge(3/2)   0           loge(3/2)
girl    0           0           loge(3/2)
both    0           0           loge(3/1)

Final vector = TF * IDF
         good             boy               girl            both                output
s1      loge(3/3)=0*1/2   loge(3/2)*1/2       0             0
s2      loge(3/3)=0*1/2   0                 loge(3/2)*1/2
s3      loge(3/3)=0*1/2   loge(3/2)*1/2   loge(3/2)*1/2     loge(3/1)*1/2

Advantages
1. Intuitive
2. Fixed size -> vocab size
3. Word important is captured

Disadvantages
1. Sparse matrix is stil exists
2. Out of Vocabulary
